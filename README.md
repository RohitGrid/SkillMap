<h2>ℹ️ Overview</h2>
<p><b>SkillMap</b> is an intelligent career development platform designed to guide users from learning new skills to landing jobs. It leverages Machine Learning to generate <b>customized roadmaps</b> based on user goals, current skill level, and industry demand.</p>

<p>Beyond learning paths, SkillMap acts as a <b>complete career assistant</b> by helping users prepare for interviews through AI-generated questions, build resumes and cover letters, and explore real-world job opportunities.</p>

<h3>🌍 Why SkillMap?</h3>
<ul>
  <li>Not knowing <i>what to learn next</i></li>
  <li>Lack of structured preparation</li>
  <li>Difficulty in creating resumes and applying for jobs</li>
</ul>

<p>SkillMap solves this by combining <b>learning + preparation + job discovery</b> into one seamless experience.</p>

<h2>🧱 Tech Stack</h2>
<table>
  <tr><td><b>Frontend</b></td><td>Next.js (App Router) · Tailwind CSS · shadcn/ui</td></tr>
  <tr><td><b>Backend</b></td><td>Next.js API Routes · Server Actions</td></tr>
  <tr><td><b>Database</b></td><td>PostgreSQL with Prisma ORM</td></tr>
  <tr><td><b>AI Layer</b></td><td>LLM APIs for roadmap, resume, interview & insights generation</td></tr>
  <tr><td><b>Authentication</b></td><td>NextAuth (middleware-based auth flow)</td></tr>
  <tr><td><b>Data Visualization</b></td><td>Charts & analytics for industry insights</td></tr>
  <tr><td><b>Deployment</b></td><td>Vercel / Netlify support</td></tr>
</table>
